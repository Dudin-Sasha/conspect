# PTC Vuforia Engine

Vuforia Engine — это отраслевой стандарт для создания
приложений дополненной реальности (AR) на основе компьютерного
зрения. Его главная задача в Unity — надежно «привязать» цифровой
контент к физическому миру, предоставляя разработчику (это SDK для создания AR окружения)


портале разработчика Vuforia, где создаются базы данных целей (Target Databases).

Image Target: Классический маркер. Вы загружаете изображение на портал. Vuforia анализирует его(feature points). B Unity Вы импортируете базу данных и добавляете на сцену prefab Image Target. Камера устройства находит в реальном мире это изображение и весь этот цифровой 'слой' появляется точно на его месте, сохраняя ПОЗИЦИЮ и перспективу при движении.


3D-объект (Model Target): Позволяет привязывать контент не к плоской картинке, а к объемному предмету (например, к промышленному двигателю, игрушке, учебному макету)

Плоские поверхности - обеспечивают без маркерное размещение, vuforia использует slam. В юнити это реализуется через компоненты PlaneFinder и ContentPositioningBehavior

### Model target

model target: Device(для распознавания с мобильного устройства) и Guid VIew(пошаговая инструкция где контент появляется при наведении на нужный ракурс объекта)

Area Target: позволяет создать и использовать цифровую карту помещения 


# Microsoft mixed reality Toolkit(MRTK)
MRTK - кроссплатформенный фреймворк для **взаимодействия**. набор скриптов префабов и инструментов для ускорения разработки на hololens, и гарнитуры mixed reality

Он абстрагирует различия между устройствами через систему профилей
- Вы настраиваете интерактивные элементы(кнопки меню и слайдеры один раз используя префабы MRTK)
- В зависимости от выбранного в настройках профиля(hololens, meta quest, и др) он настраивает элементы автоматически 

библиотека UI 
- Библиотека префабов UI
- Инструменты прототипирования
